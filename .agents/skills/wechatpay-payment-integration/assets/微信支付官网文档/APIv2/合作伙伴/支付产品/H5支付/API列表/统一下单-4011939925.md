## 应用场景

H5支付是指商户在微信客户端外的移动端网页展示商品或服务，用户在前述页面确认使用微信支付时，商户发起本服务呼起微信客户端进行支付。 主要用于触屏版的手机浏览器请求微信支付的场景。可以方便的从外部浏览器唤起微信支付。

## 接口链接

URL地址：https://api.mch.weixin.qq.com/pay/unifiedorder

## 是否需要证书

否
