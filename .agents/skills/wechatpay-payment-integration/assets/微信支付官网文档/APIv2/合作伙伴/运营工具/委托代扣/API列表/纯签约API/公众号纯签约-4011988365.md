>更新时间：2026.01.20

商户可以通过请求此接口唤起微信委托代扣的页面。

注意：

1、商户需要在前端跳转到"微信委托代扣的页面"，在商户的前端页面<head>处需要添加声明<meta name="referrer" content="no-referrer-when-downgrade">

2、此外，不能使用window.location.replace(redirect\_url)的方式跳转，而要通过window.location.href = redirect\_url的方式跳转。



## 接口说明

适用对象：服务商

请求URL： https://api.mch.weixin.qq.com/papay/partner/entrustweb

请求方式： GET

签名方式：MD5

是否需要证书： 否

## 请求参数

| 参数名 | 变量 | 类型\[长度限制\] | 必填 | 描述 |
| --- | --- | --- | --- | --- |
| 应用ID | appid | string\[1,32\] | 是 | 服务商申请的公众号或移动应用appid。<br>示例值：wxcbda96de0b165486 |
| 商户号 | mch\_id | string\[1,32\] | 是 | 商户号是商户在微信申请微信支付成功后分配的账号ID，登录平台为pay.weixin.qq.com<br>示例值：1200009811 |
| 子商户应用ID | sub\_appid | string\[1,32\] | 否 | 子商户号绑定的appid(非必填，如需操作，需要服务商在商户平台为子商户绑定)<br>示例值：wxcbda96de0b165489 |
| 子商户号 | sub\_mch\_id | string\[1,32\] | 是 | 微信支付分配的子商户号<br>示例值：1900000109 |
| 模板id | plan\_id | int | 是 | 协议模板id，设置路径见[开发步骤](https://pay.weixin.qq.com/doc/v2/partner/4011988361.md)。<br>示例值：12535 |
| 签约协议号 | contract\_code | string\[1,128\] | 是 | 商户侧的签约协议号，由商户生成，只能是数字、大小写字母的描述且在同一个商户号下唯一。<br>示例值：100000 |
| 请求序列号 | request\_serial | int64 | 是 | 商户请求签约时的序列号，要求唯一性。禁止使用0开头，序列号主要用于排序，不作为查询条件，纯数字,范围不能超过int64的范围（9223372036854775807）。<br>示例值：1000 |
| 用户账户展示名称 | contract\_display\_account | string\[1,128\] | 是 | 签约用户的名称，用于用户签约页面的“开通账号”处进行展示，参数值不支持UTF8非3字节编码的字符，例如表情符号，所以请勿传微信昵称到该字段<br>示例值：微信代扣 |
| 回调通知url | notify\_url | string\[1,256\] | 是 | 用于接收签约成功消息的回调通知地址，以http或https开头，通知url必须为外网可访问的url，不能携带参数。<br>请对notify\_url参数值进行encode处理,注意是对参数值进行encode，例如参数为notify\_url=“https://weixin.qq.com”，则需要encode的内容为https://weixin.qq.com。<br>示例值：https://weixin.qq.com |
| 版本号 | version | string\[1,8\] | 是 | 固定值1.0<br>示例值：1.0 |
| 签名 | sign | string\[1,32\] | 是 | 详见[签名生成算法](https://pay.weixin.qq.com/doc/v2/partner/4011985884.md)<br>示例值：C380BEC2BFD727A4B6845133519F3AD6 |
| 时间戳 | timestamp | string\[1,10\] | 是 | 系统当前时间，10位<br>示例值：1414488825 |
| 返回web | return\_web | int | 否 | 用来控制签约页面结束后的返回路径（不传此参数，则签约完成后停留在微信内）。<br>1表示返回签约页面的referrer url。如果获取到的referrer包含路径，会通过window.location.replace()跳转，跳转referrer url时会自动带上参数from\_wxpay=1，否则会通过window.history.back()返回上一页，此时没有参数from\_wxpay=1<br>示例值：1 |

