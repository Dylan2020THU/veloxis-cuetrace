## 接口链接

URL地址：https://api.mch.weixin.qq.com/cgi-bin/mch/newcustoms/customdeclareredeclare

重点注意：

请求接口前请先在以下页面提交您的海关信息，所有你需要报关的海关信息都需要提交，且信息真实有效：http://pay.weixin.qq.com/index.php/extend/customs

## 是否需要证书

否

## 请求参数

请求方式：post 数据格式：xml 签名方式：MD5

| 字段名 | 变量名 | 必填 | 类型 | 示例值 | 说明 |
| --- | --- | --- | --- | --- | --- |
| 签名类型 | sign\_type | 否 | String(32) | MD5 | 暂只支持MD5 |
| 签名 | sign | 是 | String(32) | C380BEC2BFD727A4B6845133519F3AD6 | 签名，详见[签名生成算法](https://pay.weixin.qq.com/doc/v2/merchant/4011985891.md) |
| 公众账号ID | appid | 是 | String(32) | wxd678efh567hg6787 | 微信分配的公众账号ID |
| 商户号 | mch\_id | 是 | String(32) | 1230000109 | 微信支付分配的商户号 |
| 商户订单号 | out\_trade\_no | 二选一 | String(32) | 20150806125346 | out\_trade\_no、transaction\_id至少一个必填 |
| 微信支付订单号 | transaction\_id | 二选一 | String(28) | 1000320306201511078440737890 | out\_trade\_no、transaction\_id至少一个必填 |
| 商户子订单号 | sub\_order\_no | 否 | String(32) | 20150806125346 | 没有子订单号可不填，否则sub\_order\_no、sub\_order\_id至少一个必填 |
| 微信子订单号 | sub\_order\_id | 否 | String(32) | 1000320306201511078440737891 | 没有子订单号可不填，否则sub\_order\_no、sub\_order\_id至少一个必填 |
| 海关 | customs | 是 | String(32) | SHANGHAI | GUANGZHOU\_ZS 广州（总署版）<br>HANGZHOU\_ZS 杭州（总署版）<br>NINGBO 宁波<br>ZHENGZHOU\_BS 郑州（保税物流中心）<br>CHONGQING 重庆<br>SHANGHAI\_ZS 上海（总署版）<br>SHENZHEN 深圳<br>ZHENGZHOU\_ZH\_ZS 郑州综保（总署版）<br>TIANJIN 天津<br>ZONGSHU 总署<br>接口文档列表没有的海关，商户在商户平台新增海关备案信息时选“广州（总署）”，备案号和备案名称填商户在海关登记的信息，然后按商户平台登记的信息调用“订单附加信息提交接口”即可 |
| 商户海关备案号 | mch\_customs\_no | 是 | String(32) | 123456 | 商户在海关登记的备案号，customs非NO，此参数必填 |

举例如下：

```
<xml>
<appid><![CDATA[wxab8acb865bb16371]]></appid>
<customs><![CDATA[SHENZHEN]]></customs>
<mch_customs_no><![CDATA[440316T004]]></mch_customs_no>
<mch_id><![CDATA[1900006511]]></mch_id>
<transaction_id><![CDATA[4200000027201712197200279161]]></transaction_id>
<sign>5D98596798203B0B1D61445707F71F87</sign>
</xml>
```

注：

参数值用XML转义即可，CDATA标签用于说明数据不被XML解析器解析。
