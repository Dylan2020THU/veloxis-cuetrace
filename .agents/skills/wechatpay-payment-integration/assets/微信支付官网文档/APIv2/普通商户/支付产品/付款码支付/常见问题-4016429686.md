### Q：付款码支付，支持预授权模式吗？类似押金方式，只是冻结，不直接扣款。

A：不支持。

### Q：付款码查询openid接口与小程序登录授权接口获取的openid是否一致？

A：若两个接口使用相同appid调用，获取的openid是同一用户的唯一标识（由appid+用户身份生成），则结果一致。

### Q：服务商模式下，使用 V2 接口下载子商户交易账单正常；但子商户自行调用直连下载交易账单接口，下载同一商户号、同一天的账单时却提示“不存在”，请问原因是什么？

A：​ 服务商下载的是服务商模式账单（含子商户数据），而子商户自行下载的是直连模式账单。若该子商户当天没有直连模式的交易，自然无账单可下。请确认下载接口与交易模式保持一致。

### Q：V2 版退款 API 机构服务商是否支持直接调用？​

A：不支持。机构服务商需调用银联侧退款接口进行处理，不直接调用微信支付退款接口。

### Q：调用付款码查询 OpenID 接口时返回“未知错误”，且付款码以 15开头，是什么原因？​

A：​以 15或 16开头的付款码属于香港钱包，而当前接口仅支持中国大陆钱包。
