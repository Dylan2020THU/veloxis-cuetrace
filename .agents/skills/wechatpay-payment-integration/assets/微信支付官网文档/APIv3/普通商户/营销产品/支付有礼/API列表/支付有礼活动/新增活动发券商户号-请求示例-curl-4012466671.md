## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/paygiftactivity/activities/126002309/merchants/add \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "merchant_id_list" : [
      "100123456"
    ],
    "add_request_no" : "100002322019090134234sfdf"
  }'

```
