## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/favor/stocks/8965000/restart \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "stock_creator_mchid" : "9865000"
  }'

```
