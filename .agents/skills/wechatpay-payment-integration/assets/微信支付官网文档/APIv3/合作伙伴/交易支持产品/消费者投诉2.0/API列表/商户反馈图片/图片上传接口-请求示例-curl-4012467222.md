## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/merchant-service/images/upload \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: multipart/form-data" \
  -F file=@"/path/to/file" \
  -F meta="{
    "filename" : "filea.jpg",
    "sha256" : "hjkahkjsjkfsjk78687dhjahdajhk"
  }"
```
