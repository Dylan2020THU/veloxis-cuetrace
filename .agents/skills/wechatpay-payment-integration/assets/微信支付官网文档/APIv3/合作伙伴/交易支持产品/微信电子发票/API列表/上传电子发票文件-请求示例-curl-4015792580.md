## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/new-tax-control-fapiao/fapiao-applications/upload-fapiao-file \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: multipart/form-data" \
  -F file=@"/path/to/file" \
  -F meta="{
    "sub_mchid" : "1900000109",
    "file_type" : "PDF",
    "digest_alogrithm" : "SM3",
    "digest" : "addca90a6a290b9642dbaccffdf01e8c22aa348940b2e96754169ba08c19e5db"
  }"
```
