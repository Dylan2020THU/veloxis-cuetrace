## curl


> PATCH

修改可用门店范围

```
curl -X PATCH \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/product-coupons/1000000013/stocks/1000000013001 \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "store_scope" : "ALL",
    "out_request_no" : "12345_20250101_A3489",
    "brand_id" : "120344"
  }'

```
