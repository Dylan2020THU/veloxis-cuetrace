## curl


> POST

上传自定义券Code

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/product-coupons/1000000013/stocks/1000000013001/upload-coupon-codes \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "out_request_no" : "upload_34657_20250101_123456",
    "code_list" : [
      "code_0000001",
      "code_0000002",
      "code_0000003"
    ],
    "brand_id" : "120344"
  }'

```
