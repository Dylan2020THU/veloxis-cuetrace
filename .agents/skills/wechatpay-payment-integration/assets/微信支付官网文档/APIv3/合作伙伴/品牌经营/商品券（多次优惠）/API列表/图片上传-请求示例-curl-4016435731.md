## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/media/upload-image \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: multipart/form-data" \
  -F meta="{
    "filename" : "header.jpg",
    "sha256" : "6aa6c99ce1d04afc2668154126c607af5a680734fa119e2a096529f6d6f2c0a2"
  }" \
  -F file=@"/path/to/file"
```
