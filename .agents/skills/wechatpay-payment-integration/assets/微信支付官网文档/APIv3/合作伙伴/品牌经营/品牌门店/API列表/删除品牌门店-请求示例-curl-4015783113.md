## curl


> DELETE

```
curl -X DELETE \
  https://api.mch.weixin.qq.com/v3/brand/partner/store/brandstores/1234567890123456 \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "brand_id" : "123456789"
  }'

```
