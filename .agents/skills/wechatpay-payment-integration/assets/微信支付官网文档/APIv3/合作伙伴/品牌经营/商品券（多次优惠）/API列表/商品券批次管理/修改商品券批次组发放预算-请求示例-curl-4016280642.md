## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/product-coupons/200000001/stock-bundles/123456789/update-budget \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "out_request_no" : "BUDGET1212512514",
    "update_mode" : "MAX_COUNT",
    "current_max_count" : 1,
    "target_max_count" : 1,
    "current_max_count_per_day" : 1,
    "target_max_count_per_day" : 1,
    "brand_id" : "120344"
  }'

```
