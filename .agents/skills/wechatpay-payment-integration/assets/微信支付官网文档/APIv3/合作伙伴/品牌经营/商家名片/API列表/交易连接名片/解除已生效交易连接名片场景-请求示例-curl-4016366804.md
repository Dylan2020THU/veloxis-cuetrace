## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/brand/card/card-links/unbind-card-link \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "brand_id" : "123456",
    "payment_scene" : "MINI_PROGRAM",
    "appid" : "abcd111111",
    "card_link_mchid" : "111222333",
    "service_id" : "00005000000000548218251086296300"
  }'

```
