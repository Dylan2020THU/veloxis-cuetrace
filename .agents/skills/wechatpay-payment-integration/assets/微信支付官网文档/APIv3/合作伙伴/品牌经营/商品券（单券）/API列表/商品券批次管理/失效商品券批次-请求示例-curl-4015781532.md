## curl


> POST

失效商品券批次

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/product-coupons/1000000013/stocks/1000000013001/deactivate \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "out_request_no" : "de_34657_20250101_123456",
    "deactivate_reason" : "批次信息有误，重新创建",
    "brand_id" : "120344"
  }'

```
