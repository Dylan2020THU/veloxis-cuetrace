## curl


> POST

```

  https://api.mch.weixin.qq.com/v3/goldplan/merchants/changegoldplanstatus \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \

  -d '{
    "sub_mchid" : "1900000109",
    "operation_type" : "OPEN",
    "operation_pay_scene" : "JSAPI_AND_MINIPROGRAM"
```
