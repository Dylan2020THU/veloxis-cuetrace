## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/favor/callbacks \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "mchid" : "9856888",
    "notify_url" : "https://pay.weixin.qq.com",
    "switch" : true
  }'

```
