## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/ecommerce/account/cancel-applications/media \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: multipart/form-data" \
  -F file=@"/path/to/file" \
  -F meta="{
    "file_name" : "filea.jpg",
    "file_digest" : "b94d27b9934d3e08a52e52d7da7dabfac484efe37a5380ee9088f7ace2efcde9"
  }"
```
