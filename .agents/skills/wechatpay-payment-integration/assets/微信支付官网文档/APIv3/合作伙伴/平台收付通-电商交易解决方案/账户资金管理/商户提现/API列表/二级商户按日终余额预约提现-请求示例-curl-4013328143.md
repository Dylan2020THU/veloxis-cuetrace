## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/platsolution/ecommerce/withdraw/day-end-balance-withdraw \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "sub_mchid" : "1900000109",
    "out_request_no" : "T20240611222222222200",
    "calculate_amount_type" : "ONLY_DAY_END_BALANCE",
    "remark" : "交易提现",
    "bank_memo" : "xx平台提现",
    "notify_url" : "https://yourapp.com/notify",
    "reserve_amount" : 1
  }'

```
