## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/ecommerce/account/cancel-applications \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "sub_mchid" : "1900000109",
    "out_apply_no" : "2019061122222222122",
    "application_info" : [
      {
        "application_type" : "SP_CANCEL_ACCOUNT_APPLICATION",
        "application_media_id" : "abc123456"
      }
    ]
  }'

```
