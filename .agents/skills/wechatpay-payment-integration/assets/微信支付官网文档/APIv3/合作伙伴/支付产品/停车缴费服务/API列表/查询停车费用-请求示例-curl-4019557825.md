## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/parking/reminders/parking-fee \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "out_serial_number" : "PARK20260423001",
    "plate_number" : "粤B12345",
    "out_parking_id" : "lot_sz_tencent_001"
  }'

```
