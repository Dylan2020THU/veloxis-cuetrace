## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/parking/reminders/ext-payment \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "out_trade_no" : "4200001234567890123456789012",
    "fee_items" : [
      {
        "fee_type" : "EXT_PARKING_PAYMENT_FEE_TYPE_UNKNOWN",
        "amount" : 1
      }
    ],
    "pay_time" : 1,
    "wx_parking_lot_id" : "21232735744117624001123604298240",
    "wx_trade_no" : "4200001234567890123456789012"
  }'

```
