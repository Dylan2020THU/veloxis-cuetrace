## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/parking/reminders/application/withdraw \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "parking_lot_audit_no" : "PLA202604230001",
    "out_parking_lot_id" : "lot_sz_tencent_001"
  }'

```
