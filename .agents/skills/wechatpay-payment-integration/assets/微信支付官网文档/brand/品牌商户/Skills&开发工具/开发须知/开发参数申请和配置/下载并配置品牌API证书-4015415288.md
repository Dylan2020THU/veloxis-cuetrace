品牌API证书具体使用说明可参见可参见[开发必要参数说明](https://pay.weixin.qq.com/doc/brand/4015415289.md)

功能介绍：

品牌API证书，是指由品牌商户申请的，用来证实品牌商户身份的证书。品牌API证书由证书授权机构Certificate Authority（简称CA）颁发。证书中包含品牌商户的ID、公司名称、公钥等信息。请妥善保管API证书文件，防止泄露。

使用场景：

调用微信支付-品牌经营相关API接口时，需要用品牌API证书的私钥签名，微信支付会用品牌API证书公钥验签。

获取步骤：

管理员可登录[微信支付品牌经营平台](https://pay.weixin.qq.com/xdc/brandhomeweb/brand/home#/)，在【账号管理】->【安全中心】目录下获取证书

1、访问安全中心页面，在“品牌 API 证书”模块点击申请

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/6b48157bcbe5d6fd74b87049077774b2.png)

以下为具体下载步骤：

1.在安全中心，证书申请页面上点击“申请证书”。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/d6f73baf5e1d3a712c8c59367613fd31.png)

2.在弹出窗口内点击“下载工具”按钮下载证书工具。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/3efca0da0f1237b8ae22f38f8153fd0f.png)

3.安装证书工具并打开，选择证书需要存储的路径后点击“申请证书”。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/e3adff96a77ad32b59a4ec3e866e14f4.png)

4.在证书工具中，将复制的品牌商户信息粘贴并点击“下一步”

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/edd424719d20a301e2cf63da01e39f73.png)

5.获取请求串

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/71d0691977160179ed5c2509feb91032.png)

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/2904b01e75b6f001b959581fef3fb112.png)

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/2d60fae03ce6504e31d93b5ff6c590f1.png)

6.生成证书串

步骤1：在【品牌经营平台】-“复制证书串”环节，点击“复制证书串”按钮后；

步骤2：在【证书工具】-“复制请求串”环节，点击“下一步”按钮进入“粘贴证书串”环节；

步骤3：在【证书工具】-“粘贴证书串”环节，点击“粘贴”按钮后；

步骤4：点击“下一步”按钮，进入【证书工具】-“生成证书”环节

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/15ee3cc6bad7019efc4e020d521a0c53.png)

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/33ba816f6e474781b49206ca2b2cdbca.png)

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/d63116f0730de0376e18b51b55c02293.png)

7.在【证书工具】-“生成证书”环节，已完成申请证书流程，点击“查看证书文件夹”，查看已生成的证书文件。

8.完成
