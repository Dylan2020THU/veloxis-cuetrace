## 1.什么时候需要更换？

（1）品牌API证书丢失、泄露需要及时更换

（2）品牌API证书过期

## 2. 如何更换品牌API证书？

### 第一步：申请一张新的品牌API证书

具体步骤请参考[下载并配置品牌API证书](https://pay.weixin.qq.com/doc/brand/4015415288.md)

### 第二步：作废旧品牌API证书

1、管理员登录【微信支付品牌经营平台->  账号管理 ->  安全中心】，找到品牌API证书，点击“管理”

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/1f5590c4b0387c08f016401ad16afb78.png)

2、在API证书管理页面，点击对应证书旁边的“作废”按钮

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/f066b0a9aebc71b587b455c7aede303d.png)

3、在页面弹框中点击“确认作废”

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/b9d02154b9e463a7daac3720ea3331dd.png)

4、输入操作密码和手机验证码后，点击“确定”

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/908649948381645ef2218891083d94f1.png)
