>更新时间：2025.07.21

品牌API密钥主要用于回调信息解密，具体使用方式可参见接口规则文档中的[如何解密回调报文](https://pay.weixin.qq.com/doc/brand/4015407591.md)

1.登录[品牌经营平台](https://pay.weixin.qq.com/xdc/brandhomeweb/brand/home#/)，进入【账号管理 \> 安全中心】目录，点击【品牌API密钥】。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/faa06686882b653083073915a8c5a6e1.png)

2.输入API密钥，内容为32位字符，包括数字及大小写字母。点击确定设置获取短信验证码。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/b73dd4a671cdc916689b10b0526171c0.png)

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/5c7e89d75ed8d6063b6995171a8d0666.png)

3.输入短信验证码，点击“确认”即设置成功。

4.完成
