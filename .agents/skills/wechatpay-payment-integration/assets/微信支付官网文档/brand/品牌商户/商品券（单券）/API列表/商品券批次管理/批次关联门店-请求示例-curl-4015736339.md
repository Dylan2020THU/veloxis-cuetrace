## curl


> POST

关联门店列表

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/product-coupons/1000000013/stocks/1000000013001/associate-stores \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "store_list" : [
      {
        "store_id" : "100000001"
      },
      {
        "store_id" : "100000002"
      },
      {
        "store_id" : "100000003"
      },
      {
        "store_id" : "12"
      }
    ]
  }'

```
