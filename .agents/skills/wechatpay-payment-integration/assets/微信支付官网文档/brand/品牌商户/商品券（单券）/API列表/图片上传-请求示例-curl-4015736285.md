## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/media/upload-image \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: multipart/form-data" \
  -F meta="{
    "filename" : "header.jpg",
    "sha256" : "6aa6c99ce1d04afc2668154126c607af5a680734fa119e2a096529f6d6f2c0a2"
  }" \
  -F file=@"/path/to/file"
```
