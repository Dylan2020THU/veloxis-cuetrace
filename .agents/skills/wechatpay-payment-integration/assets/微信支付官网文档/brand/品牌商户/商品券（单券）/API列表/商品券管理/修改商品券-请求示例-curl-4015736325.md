## curl


> PATCH

调整商品券标题

```
curl -X PATCH \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/product-coupons/1000000013 \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "display_info" : {
      "name" : "全场满100立打8折-新名字",
      "image_url" : "https://wxpaylogo.qpic.cn/wxpaylogo/xxxxx/xxx",
      "background_url" : "https://wxpaylogo.qpic.cn/wxpaylogo/xxxxx/xxx",
      "detail_image_url_list" : [
        "https://wxpaylogo.qpic.cn/wxpaylogo/xxxxx/xxx"
      ]
    },
    "out_request_no" : "12345_20250101_A3489"
  }'

```
