## curl


> POST

调整批次发放次数总上限

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/product-coupons/1000000013/stocks/1000000013001/update-budget \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "target_max_count" : 200,
    "update_mode" : "MAX_COUNT",
    "out_request_no" : "update_budget_34657_20250101_123456",
    "current_max_count" : 10000000
  }'

```
