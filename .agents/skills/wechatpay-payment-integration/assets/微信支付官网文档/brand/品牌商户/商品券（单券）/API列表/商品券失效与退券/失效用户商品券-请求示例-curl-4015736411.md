## curl


> POST

使用户券失效

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/users/oh-394z-6CGkNoJrsDLTTUKiAnp4/coupons/Code_123456/deactivate \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "product_coupon_id" : "1000000013",
    "appid" : "wx233544546545989",
    "deactivate_reason" : "商品已下线，使用户商品券失效",
    "stock_id" : "1000000013001",
    "out_request_no" : "MCHDEACTIVATE202003101234"
  }'

```
