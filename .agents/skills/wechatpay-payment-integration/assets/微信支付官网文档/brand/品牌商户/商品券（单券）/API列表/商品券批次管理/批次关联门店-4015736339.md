>更新时间：2025.08.28

品牌方可以通过该接口将品牌的门店列表与商品券批次关联。

前置条件：

1. 已创建 `usage_mode` 不为 `PROGRESSIVE_BUNDLE` 的商品券对应的批次

2. 批次的 `store_scope` 为 `SPECIFIC`

3. 批次未失效或过期

4. `usage_mode` 为 `PROGRESSIVE_BUNDLE` 的批次在批次组内，不可使用本接口，请使用【[批次组批量关联门店API](https://pay.weixin.qq.com/doc/brand/4016279709.md)\]





## 接口说明

支持商户：【品牌商户】

请求方式：【POST】 `/brand/marketing/product-coupon/product-coupons/{product_coupon_id}/stocks/{stock_id}/associate-stores`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/brand/4015596635.md)

接口限频：20/秒（品牌ID维度）

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/brand/4015407568.md) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `application/json` |
| Wechatpay-Serial | string | true | 【微信支付公钥ID】  请传入brand\_id对应的微信支付公钥ID，接口将会校验两者的关联关系，参考 [微信支付公钥产品简介及使用说明](https://pay.weixin.qq.com/doc/brand/4015597001.md) 获取微信支付公钥ID和相关的介绍。以下两种场景将使用到微信支付公钥：<br>1、接收到接口的返回内容，需要使用微信支付公钥进行验签；<br>2、调用含有敏感信息参数（如姓名、身份证号码）的接口时，需要使用微信支付公钥加密敏感信息后再传输参数，加密指引请参考 [微信支付公钥加密敏感信息指引](https://pay.weixin.qq.com/doc/brand/4015407587.md)。 |
### Path 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| product_coupon_id | string | true | 【商品券ID】 商品券的唯一标识， [创建商品券](https://pay.weixin.qq.com/doc/brand/4015736297.md) 时由微信支付生成 |
| stock_id | string | true | 【批次ID】 商品券批次的唯一标识，商品券批次创建时由微信支付生成（可使用【 [创建商品券API](https://pay.weixin.qq.com/doc/brand/4015736297.md)】或【 [添加商品券批次API](https://pay.weixin.qq.com/doc/brand/4015736342.md)】创建），请确保该批次属于 `product_coupon_id` 对应的商品券，且 `store_scope` 为 `SPECIFIC`，否则无法关联门店 |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| store_list | array[object] | true | 【门店列表】 商户欲关联的门店列表，<br>单次最多可批量关联200个门店 |
| store_list.store_id | string | true | 【门店ID】 门店的唯一标识，调用【 [创建品牌门店API](https://pay.weixin.qq.com/doc/brand/4015782988.md)】时由微信支付生成 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| total_count | integer | true | 【去重后门店总数】 系统会自动对请求中的门店去重，这里是去重后的门店总数 |
| success_store_list | array[object] | false | 【关联成功的门店列表】 关联成功的门店列表 |
| success_store_list.store_id | string | true | 【门店ID】 门店的唯一标识，调用【 [创建品牌门店API](https://pay.weixin.qq.com/doc/brand/4015782988.md)】时由微信支付生成 |
| failed_store_list | array[object] | false | 【关联失败的门店列表】 每个关联失败门店的详细错误信息 |
| failed_store_list.store_id | string | true | 【门店ID】 关联失败的门店ID |
| failed_store_list.code | string(40) | true | 【失败错误码】 门店关联失败的错误码 |
| failed_store_list.message | string(200) | true | 【失败错误信息】 门店关联失败的错误描述 |
