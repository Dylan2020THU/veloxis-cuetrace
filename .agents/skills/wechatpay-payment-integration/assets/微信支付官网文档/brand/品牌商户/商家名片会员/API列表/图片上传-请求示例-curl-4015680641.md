## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/card-member/media/image-upload \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: multipart/form-data" \
  -F file=@"/path/to/file" \
  -F meta="{
    "filename" : "file.jpg",
    "file_digest" : "hjkahkjsjkfsjk78687dhjahdajhk"
  }"
```
