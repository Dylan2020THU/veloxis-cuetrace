## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/card-member/user-cards/478515832665/invalidate \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "card_id" : "pbLatjvWOibDc5-TBnbUk1pD12o0",
    "openid" : "obLatjnx9gnqzS4myYGmLZ7LgLBA",
    "invalid_reason" : "传入的自定义作废原因"
  }'

```
