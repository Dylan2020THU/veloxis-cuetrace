>更新时间：2025.11.19

生成用于商家小程序或 H5 拉起入会组件的凭证




## 接口说明

支持商户：【品牌商户】

请求方式：【POST】 `/brand/card-member/pre-auth-tokens`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/brand/4015596635.md)

接口限频：10/秒（品牌ID维度）

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/brand/4015407568.md) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `application/json` |
| Wechatpay-Serial | string | true | 【微信支付公钥ID】  请传入brand\_id对应的微信支付公钥ID，接口将会校验两者的关联关系，参考 [微信支付公钥产品简介及使用说明](https://pay.weixin.qq.com/doc/brand/4015597001.md) 获取微信支付公钥ID和相关的介绍。以下两种场景将使用到微信支付公钥：<br>1、接收到接口的返回内容，需要使用微信支付公钥进行验签；<br>2、调用含有敏感信息参数（如姓名、身份证号码）的接口时，需要使用微信支付公钥加密敏感信息后再传输参数，加密指引请参考 [微信支付公钥加密敏感信息指引](https://pay.weixin.qq.com/doc/brand/4015407587.md)。 |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| card_id | string(32) | true | 【会员卡模板 ID】 商家 [创建会员卡模板](https://pay.weixin.qq.com/doc/brand/4015582711.md) 成功后系统返回的会员卡模板ID |
| openid | string(128) | true | 【用户标识】 用户在品牌商家会员卡模板AppID下的唯一标识，获取方式详见 [参数说明](https://pay.weixin.qq.com/doc/brand/4015407536.md#OpenID)。 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| token | string(32) | true | 【预授权 token】 用于H5 和小程序场景拉起会员开卡组件投放会员卡 |
| expire_time | string | true | 【token 过期时间】 token过期时间（生成后一小时有效），需遵循 [RFC3339](https://datatracker.ietf.org/doc/html/rfc3339) 标准格式：yyyy-MM-DDTHH:mm:ss+TIMEZONE。yyyy-MM-DD 表示年月日；T 字符用于分隔日期和时间部分；HH:mm:ss 表示具体的时分秒；TIMEZONE 表示时区（例如，+08:00 对应东八区时间，即北京时间）。示例：2015-05-20T13:29:35+08:00 表示北京时间2015年5月20日13点29分35秒。 |
