## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/store/brandstores/1234567890123456/unbindrecipient \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "mchid" : "1230000109"
  }'

```
