>更新时间：2025.08.04

给品牌门店绑定收款商户号。绑定成功后，商户号可为门店收款。一个门店目前最多只能绑定三个收款商户号。




## 接口说明

支持商户：【品牌商户】

请求方式：【POST】 `/brand/store/brandstores/{store_id}/bindrecipient`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/brand/4015596635.md)

接口限频：900/分钟（品牌ID维度）

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/brand/4015407568.md) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `application/json` |
| Wechatpay-Serial | string | true | 【微信支付公钥ID】  请传入brand\_id对应的微信支付公钥ID，接口将会校验两者的关联关系，参考 [微信支付公钥产品简介及使用说明](https://pay.weixin.qq.com/doc/brand/4015597001.md) 获取微信支付公钥ID和相关的介绍。以下两种场景将使用到微信支付公钥：<br>1、接收到接口的返回内容，需要使用微信支付公钥进行验签；<br>2、调用含有敏感信息参数（如姓名、身份证号码）的接口时，需要使用微信支付公钥加密敏感信息后再传输参数，加密指引请参考 [微信支付公钥加密敏感信息指引](https://pay.weixin.qq.com/doc/brand/4015407587.md)。 |
### Path 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| store_id | string | true | 【品牌门店ID】 创建品牌门店后，系统为该门店分配的唯一ID。 |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| mchid | string(16) | true | 【门店收款商户号】  门店的收款商户号，仅允许填入品牌已关联的商户号，若未关联请先前往“品牌经营平台-账号管理-收款商户号管理”添加商户号。 |
| company_name | string(256) | true | 【门店收款主体】 门店收款的主体信息，支持企业，个体户，小微。 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| store_id | string | false | 【品牌门店ID】 创建品牌门店后，系统为该门店分配的唯一ID。 |
| mchid | string(16) | false | 【门店收款商户号】 门店的收款商户号，仅支持绑定品牌已关联的商户号。 |
| company_name | string(256) | false | 【门店收款主体】 门店收款的主体信息，支持企业，个体户，小微。 |
| recipient_state | string | false | 【收款绑定状态】 门店收款商户号的绑定状态<br>可选取值<br><ul><li>`CONFIRMED`:  门店已绑定此收款商户号</li><li>`ADMIN_REJECTED`:  商户管理员已拒绝</li><li>`CONFIRMING`:  商户号管理员确认中</li><li>`TIMEOUT_REJECTED`:  由于商户管理员确认超时，系统自动拒绝此绑定</li></ul> |
