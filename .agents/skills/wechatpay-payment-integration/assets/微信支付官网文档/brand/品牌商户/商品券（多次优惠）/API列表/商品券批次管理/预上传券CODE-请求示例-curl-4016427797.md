## curl


> POST

上传自定义券Code

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/product-coupons/1000000013/stocks/1000000013001/upload-coupon-codes \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "out_request_no" : "upload_34657_20250101_123456",
    "code_list" : [
      "code_0000001",
      "code_0000002",
      "code_0000003"
    ]
  }'

```
