## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/product-coupons/200000001/stock-bundles/123456789/update-budget \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "out_request_no" : "BUDGET1212512514",
    "update_mode" : "MAX_COUNT",
    "current_max_count" : 1,
    "target_max_count" : 1,
    "current_max_count_per_day" : 1,
    "target_max_count_per_day" : 1
  }'

```
