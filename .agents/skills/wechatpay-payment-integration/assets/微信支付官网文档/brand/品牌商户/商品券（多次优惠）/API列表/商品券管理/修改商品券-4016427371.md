>更新时间：2025.11.06

品牌方可以通过该接口修改商品券信息。

注意： 修改只会对新发的券生效，历史已经发放给用户的券不会改变。

前置条件：商品券处于 `EFFECTIVE` 状态




## 接口说明

支持商户：【品牌商户】

请求方式：【PATCH】 `/brand/marketing/product-coupon/product-coupons/{product_coupon_id}`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/brand/4015596635.md)

接口限频：20/秒（品牌ID维度）
