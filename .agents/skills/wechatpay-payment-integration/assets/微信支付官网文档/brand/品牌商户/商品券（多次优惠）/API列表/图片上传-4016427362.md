>更新时间：2025.11.06

商品券接口调用过程中需要传入各种图片URL，请使用本接口上传图片到微信支付，并使用接口返回的URL。




## 接口说明

支持商户：【品牌商户】

请求方式：【POST】 `/brand/marketing/product-coupon/media/upload-image`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/brand/4015596635.md)

接口限频：20/秒（品牌ID维度）

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/brand/4015407568.md) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `multipart/form-data` |
| Wechatpay-Serial | string | true | 【微信支付公钥ID】  请传入brand\_id对应的微信支付公钥ID，接口将会校验两者的关联关系，参考 [微信支付公钥产品简介及使用说明](https://pay.weixin.qq.com/doc/brand/4015597001.md) 获取微信支付公钥ID和相关的介绍。以下两种场景将使用到微信支付公钥：<br>1、接收到接口的返回内容，需要使用微信支付公钥进行验签；<br>2、调用含有敏感信息参数（如姓名、身份证号码）的接口时，需要使用微信支付公钥加密敏感信息后再传输参数，加密指引请参考 [微信支付公钥加密敏感信息指引](https://pay.weixin.qq.com/doc/brand/4015407587.md)。 |
### Form 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| meta | object | true | 【媒体图片文件元信息】 使用 JSON 格式传输 |
| meta.filename | string(150) | true | 【图片文件名】 商户上传的媒体图片的名称，商户自定义，必须以JPG、BMP、PNG为后缀，名称不能超过150个字符 |
| meta.sha256 | string | true | 【图片文件摘要】 图片文件的文件摘要，即对图片文件的二进制内容进行sha256计算得到的值 |
| file | file | true | 【媒体图片二进制内容】 将媒体图片进行二进制转换，得到的媒体图片二进制内容，在请求Body中上传此二进制内容。媒体图片只支持JPG、BMP、PNG格式，文件大小不能超过2M。 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| image_url | string(256) | true | 【图片URL】 微信返回的媒体文件标识url。有效期为永久 |

| --- | --- | --- | --- |

| 400 | INVALID\_REQUEST | HTTP 请求不符合微信支付 APIv3 接口规则 | 请参阅 [接口规则](https://pay.weixin.qq.com/doc/brand/4015407525.md) |

| 500 | SYSTEM\_ERROR | 系统异常，请稍后重试 | 请稍后重试 |
| 403 | NO\_AUTH | 品牌没有此接口权限 | 品牌没有此接口权限 |
| 400 | INVALID\_REQUEST | 参数中的 meta 字段与文件的基本信息不一致，请确认文件名是否一致 | 参数中的 meta 字段与文件的基本信息不一致，请确认文件名是否一致 |
| 400 | PARAM\_ERROR | 参数错误，上传图片为空 | 请提供有效图片二进制 |
| 400 | PARAM\_ERROR | 参数错误，上传的图片格式未知或不在支持范围内 | 请使用文档中指定类型的图片 |
| 400 | INVALID\_REQUEST | 参数错误，上传的图片与 meta 中的 sha256 值校验不通过 | 请使用正确的sha256 值 |
| 400 | PARAM\_ERROR | 参数错误，上传的图片过大 | 请缩小图片文件的体积 |
| 400 | PARAM\_ERROR | Http header中缺少必填参数Wechatpay-Serial | 缺少必要商品券图片，请补充上传后重试 |
