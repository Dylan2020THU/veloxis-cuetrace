## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/brand/marketing/product-coupon/users/oh-394z-6CGkNoJrsDLTTUKiAnp4/coupon-bundles/123446565767/deactivate \
  -H "Authorization: WECHATPAY-BRAND-SHA256-RSA2048 brand_id=\"XXXX\",..." \
  -H "Accept: application/json" \
  -H "Wechatpay-Serial: PUB_KEY_ID_XXXX"  \
  -H "Content-Type: application/json" \
  -d '{
    "product_coupon_id" : "1002323",
    "stock_bundle_id" : "100232301",
    "appid" : "wx233544546545989",
    "out_request_no" : "1002600620019090123144054436",
    "deactivate_reason" : "商品已下线，使用户商品券失效"
  }'

```
