## Retained platform identifiers

`payerOpenid` remains only where a signed WeChat payment, contract, refund, or callback must carry or compare the platform payer parameter. It never authorizes order lookup, refund access, ownership, or cross-account operations.

Independent administrator OPENID remains confined to the administrator trust boundary and its binding records. Administrator actions address ordinary users and business records by `accountId`; administrator OPENID is never a substitute for a user account foreign key.
