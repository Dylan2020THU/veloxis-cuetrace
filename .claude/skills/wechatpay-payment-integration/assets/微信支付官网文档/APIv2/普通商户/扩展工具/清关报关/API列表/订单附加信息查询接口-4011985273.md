>更新时间：2024.11.13

## 应用场景

商户通过订单号查询提交的订单附加信息。如果是微信收集的实名信息，查询接口不返回实名信息内容

## 接口链接

URL地址：https://api.mch.weixin.qq.com/cgi-bin/mch/customs/customdeclarequery

## 是否需要证书

否
