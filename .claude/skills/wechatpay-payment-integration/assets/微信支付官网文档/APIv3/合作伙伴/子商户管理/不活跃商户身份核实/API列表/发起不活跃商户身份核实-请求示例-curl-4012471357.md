## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/compliance/inactive-merchant-identity-verification/merchants \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "sub_mchid" : "1900000000"
  }'

```
