## curl


> POST

取消关联门店列表

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/product-coupons/1000000013/stocks/1000000013001/disassociate-stores \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "store_list" : [
      {
        "store_id" : "100000001"
      },
      {
        "store_id" : "100000002"
      },
      {
        "store_id" : "100000003"
      },
      {
        "store_id" : "12"
      }
    ],
    "brand_id" : "120344"
  }'

```
