## curl


> POST

确认发放用户券

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/users/oh-394z-6CGkNoJrsDLTTUKiAnp4/coupons/Code_123456/confirm \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "product_coupon_id" : "1000000013",
    "appid" : "wx233544546545989",
    "stock_id" : "1000000013001",
    "out_request_no" : "MCHCONFIRM202003101234",
    "brand_id" : "120344"
  }'

```
