## curl


> POST

发送多次优惠批次组

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/users/oh-394z-6CGkNoJrsDLTTUKiAnp4/coupon-bundles \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "stock_bundle_id" : "712315129419284901",
    "product_coupon_id" : "1000000014",
    "appid" : "wx233544546545989",
    "send_request_no" : "MCHCONFIRM202003101234",
    "brand_id" : "120344"
  }'

```
