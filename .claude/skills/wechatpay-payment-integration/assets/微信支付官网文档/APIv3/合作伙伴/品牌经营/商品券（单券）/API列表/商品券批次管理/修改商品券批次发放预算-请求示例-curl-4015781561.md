## curl


> POST

调整批次发放次数总上限

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/product-coupons/1000000013/stocks/1000000013001/update-budget \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "target_max_count" : 200,
    "update_mode" : "MAX_COUNT",
    "out_request_no" : "update_budget_34657_20250101_123456",
    "current_max_count" : 10000000,
    "brand_id" : "120344"
  }'

```
