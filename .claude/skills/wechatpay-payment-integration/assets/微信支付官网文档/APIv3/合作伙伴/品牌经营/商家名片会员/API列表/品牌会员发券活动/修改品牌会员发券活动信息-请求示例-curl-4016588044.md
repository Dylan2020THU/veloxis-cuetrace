## curl


> PATCH

```

  https://api.mch.weixin.qq.com/v3/brand/partner/card-member/membership-activities/25 \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "brand_id" : "1004",
    "title" : "入会享券",
    "sub_title" : "5张满减券",
    "begin_time" : "2025-05-20T13:29:35.120+08:00",
    "end_time" : "2026-05-20T13:29:35.120+08:00"
  }'

```
