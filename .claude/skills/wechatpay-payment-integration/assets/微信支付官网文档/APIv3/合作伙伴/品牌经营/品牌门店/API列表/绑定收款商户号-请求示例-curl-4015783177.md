## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/brand/partner/store/brandstores/1234567890123456/bindrecipient \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "brand_id" : "123456789",
    "mchid" : "1230000109",
    "company_name" : "腾讯科技（深圳）有限公司"
  }'

```
