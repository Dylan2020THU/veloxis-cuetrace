## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/brand/card/card-links/cancel-applyment \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "business_code" : "1900013511_10000",
    "brand_id" : "123456"
  }'

```
