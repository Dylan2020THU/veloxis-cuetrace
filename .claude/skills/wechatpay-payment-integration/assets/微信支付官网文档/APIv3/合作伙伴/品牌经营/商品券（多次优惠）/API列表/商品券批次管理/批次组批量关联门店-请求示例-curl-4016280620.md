## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partner/product-coupon/product-coupons/200000001/stock-bundles/123456789/associate-stores \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "store_list" : [
      {
        "store_id" : "100000001"
      }
    ],
    "brand_id" : "120344"
  }'

```
