## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/favor/users/openid_example/coupons \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "stock_id" : "example_stock_id",
    "out_request_no" : "example_out_request_no",
    "appid" : "example_appid",
    "stock_creator_mchid" : "example_stock_creator_mchid",
    "coupon_value" : 1,
    "coupon_minimum" : 1
  }'

```
