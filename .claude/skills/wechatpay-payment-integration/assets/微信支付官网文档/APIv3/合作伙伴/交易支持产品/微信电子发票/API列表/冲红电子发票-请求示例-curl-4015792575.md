## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/new-tax-control-fapiao/fapiao-applications/4200000444201910177461284488/reverse \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "sub_mchid" : "1900000109",
    "reverse_reason" : "非数电模式：退款；数电模式：ISSUE_ERROR|SALES_RETURN|SERVICE_SUSPENSION|SALES_DISCOUNT",
    "fapiao_information" : [
      {
        "fapiao_id" : "20200701123456",
        "fapiao_number" : "12897794"
      }
    ]
  }'

```
