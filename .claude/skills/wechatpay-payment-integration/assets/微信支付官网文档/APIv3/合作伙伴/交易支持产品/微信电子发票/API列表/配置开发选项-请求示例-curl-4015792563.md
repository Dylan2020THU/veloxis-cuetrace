## curl


> PATCH

```
curl -X PATCH \
  https://api.mch.weixin.qq.com/v3/new-tax-control-fapiao/merchant/development-config \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "callback_url" : "https://pay.weixin.qq.com/callback",
    "sub_mch_code" : "1900000109",
    "show_fapiao_cell" : false,
    "support_vat_fapiao" : false
  }'

```
