## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/merchant-service/complaints-v2/200201820200101080076610000/update-refund-progress \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "action" : "APPROVE",
    "launch_refund_day" : 3,
    "reject_reason" : "拒绝退款",
    "reject_media_list" : [
      "aabbccdd"
    ],
    "remark" : "已处理完成"
  }'

```
