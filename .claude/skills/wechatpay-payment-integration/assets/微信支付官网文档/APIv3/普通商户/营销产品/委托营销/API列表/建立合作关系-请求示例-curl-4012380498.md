## curl


> POST

```
curl -X POST \
  https://api.mch.weixin.qq.com/v3/marketing/partnerships/build \
  -H "Idempotency-Key: buildreq_001" \
  -H "Authorization: WECHATPAY2-SHA256-RSA2048 mchid=\"1900000001\",..." \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "partner" : {
      "type" : "APPID",
      "appid" : "wx4e1916a585d1f4e9",
      "merchant_id" : "2480029552"
    },
    "authorized_data" : {
      "business_type" : "FAVOR_STOCK",
      "stock_id" : "2433405"
    }
  }'

```
